Науковий керівник &#8211; професор Помірко Р.С.</p><p>З 2016 року працює на посаді доцента кафедри французької філології Львівського національного університету імені Івана Франка.</p></div></section><section class='section'><h2>Методичні матеріали</h2><div><p>Лесько Х. С., І.
