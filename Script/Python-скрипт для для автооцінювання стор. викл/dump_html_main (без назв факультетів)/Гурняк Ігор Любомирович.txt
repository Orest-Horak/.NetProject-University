Дисертацію захистив у спеціалізованій вченій раді Української академії Зовнішньої торгівлі Міністерства економіки та з питань європейської інтеграції України.</p><p>Основні етапи педагогічної діяльності у вищих навчальних закладах освіти:</p><p>01.09.2001 р. &#8211; 01.02.2011 р. &#8211; асистент кафедри міжнародного економічного аналізу і фінансів,</p><p>01.02.2011 р. &#8211; по теперішній час &#8211; доцент кафедри міжнародного економічного аналізу і фінансів.</p><p>1999 р. – 2007 р. – робота у Львівській торгово-промисловій палаті.</p><p>Веде лекційні курси і проводить практичні заняття з дисциплін: -&#8220;Мікроекономіка&#8221; для студентів другого курсу факультету міжнародних відносин;</p><p>-&#8220;Актуальні проблеми фінансового менеджменту&#8221; для студентів освітньо-кваліфікаційного рівня «магістр» факультету міжнародних відносин;</p><p>-&#8220;Основи аудиту&#8221; для студентів четвертого курсу факультету міжнародних відносин;</p><p>&#8211; &#8220;Облік та аналіз зовнішньоекономічної діяльності&#8221; для студентів другого курсу факультету міжнародних відносин;</p><p>&#8211; &#8220;Фінансовий аналіз&#8221; для студентів четвертого курсу факультету міжнародних відносин, для студентів освітньо-кваліфікаційного рівня «магістр» факультету міжнародних відносин;</p><p>&#8211; &#8220;Сучасні проблеми фінансового обліку&#8221;
