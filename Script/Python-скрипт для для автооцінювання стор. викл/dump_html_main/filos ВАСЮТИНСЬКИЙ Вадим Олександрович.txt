Серед найвагоміших наукових результатів — створення інтеракційної теорії влади, в якій всебічно висвітлено соціально-психологічну природу влади як <a title="Дискурс" href="https://uk.wikipedia.org/wiki/%D0%94%D0%B8%D1%81%D0%BA%D1%83%D1%80%D1%81">дискурсу</a> інтерсуб’єктної взаємодії, досліджено особистісні, вікові, гендерні, міжстатеві, сімейні, політико-ідеологічні виміри владно-підвладних взаємин. В. О. Васютинський досліджував проблеми психології спільнот, психології бідності як стилю життя, психології масової свідомості та поведінки, проблеми мовно-етнічної ідентичності. Серед його наукових інтересів — теорія і практика <a title="Психотерапія" href="https://uk.wikipedia.org/wiki/%D0%9F%D1%81%D0%B8%D1%85%D0%BE%D1%82%D0%B5%D1%80%D0%B0%D0%BF%D1%96%D1%8F">психотерапії</a>, він є сертифікованим фахівцем із<a class="new" title="Груповий психоаналіз (ще не написана)" href="https://uk.wikipedia.org/w/index.php?title=%D0%93%D1%80%D1%83%D0%BF%D0%BE%D0%B2%D0%B8%D0%B9_%D0%BF%D1%81%D0%B8%D1%85%D0%BE%D0%B0%D0%BD%D0%B0%D0%BB%D1%96%D0%B7&amp;action=edit&amp;redlink=1">групового психоаналізу</a>.</p><p>В. О. Васютинський є членом <a class="new" title="Товариство психологів України (ще не написана)" href="https://uk.wikipedia.org/w/index.php?title=%D0%A2%D0%BE%D0%B2%D0%B0%D1%80%D0%B8%D1%81%D1%82%D0%B2%D0%BE_%D0%BF%D1%81%D0%B8%D1%85%D0%BE%D0%BB%D0%BE%D0%B3%D1%96%D0%B2_%D0%A3%D0%BA%D1%80%D0%B0%D1%97%D0%BD%D0%B8&amp;action=edit&amp;redlink=1">Товариства психологів України</a> (з 1989), <a class="new" title="Асоціація політичних психологів України (ще не написана)" href="https://uk.wikipedia.org/w/index.php?title=%D0%90%D1%81%D0%BE%D1%86%D1%96%D0%B0%D1%86%D1%96%D1%8F_%D0%BF%D0%BE%D0%BB%D1%96%D1%82%D0%B8%D1%87%D0%BD%D0%B8%D1%85_%D0%BF%D1%81%D0%B8%D1%85%D0%BE%D0%BB%D0%BE%D0%B3%D1%96%D0%B2_%D0%A3%D0%BA%D1%80%D0%B0%D1%97%D0%BD%D0%B8&amp;action=edit&amp;redlink=1">Асоціації політичних психологів України</a> (з 1995), <a title="Українська спілка психотерапевтів" href="https://uk.wikipedia.org/wiki/%D0%A3%D0%BA%D1%80%D0%B0%D1%97%D0%BD%D1%81%D1%8C%D0%BA%D0%B0_%D1%81%D0%BF%D1%96%D0%BB%D0%BA%D0%B0_%D0%BF%D1%81%D0%B8%D1%85%D0%BE%D1%82%D0%B5%D1%80%D0%B0%D0%BF%D0%B5%D0%B2%D1%82%D1%96%D0%B2">Української спілки психотерапевтів</a> (з 2000), Української асоціації організаційних психологів і психологів праці (з 2002), Польської асоціації соціальної психології (з 2011).</p></div></section><section В 1977 році закінчив відділення психології філософського факультету <a title="Київський національний університет імені Тараса Шевченка" href="https://uk.wikipedia.org/wiki/%D0%9A%D0%B8%D1%97%D0%B2%D1%81%D1%8C%D0%BA%D0%B8%D0%B9_%D0%BD%D0%B0%D1%86%D1%96%D0%BE%D0%BD%D0%B0%D0%BB%D1%8C%D0%BD%D0%B8%D0%B9_%D1%83%D0%BD%D1%96%D0%B2%D0%B5%D1%80%D1%81%D0%B8%D1%82%D0%B5%D1%82_%D1%96%D0%BC%D0%B5%D0%BD%D1%96_%D0%A2%D0%B0%D1%80%D0%B0%D1%81%D0%B0_%D0%A8%D0%B5%D0%B2%D1%87%D0%B5%D0%BD%D0%BA%D0%B0">Київського державного університету ім. Т. Г. Шевченка</a>.</p><p>Працював психологом <a title="Львівська обласна клінічна психіатрична лікарня" href="https://uk.wikipedia.org/wiki/%D0%9B%D1%8C%D0%B2%D1%96%D0%B2%D1%81%D1%8C%D0%BA%D0%B0_%D0%BE%D0%B1%D0%BB%D0%B0%D1%81%D0%BD%D0%B0_%D0%BA%D0%BB%D1%96%D0%BD%D1%96%D1%87%D0%BD%D0%B0_%D0%BF%D1%81%D0%B8%D1%85%D1%96%D0%B0%D1%82%D1%80%D0%B8%D1%87%D0%BD%D0%B0_%D0%BB%D1%96%D0%BA%D0%B0%D1%80%D0%BD%D1%8F">Львівської обласної психіатричної лікарні</a> (1977-1980), асистентом кафедри педагогіки та психології <a title="Львівський національний університет імені Івана Франка" href="https://uk.wikipedia.org/wiki/%D0%9B%D1%8C%D0%B2%D1%96%D0%B2%D1%81%D1%8C%D0%BA%D0%B8%D0%B9_%D0%BD%D0%B0%D1%86%D1%96%D0%BE%D0%BD%D0%B0%D0%BB%D1%8C%D0%BD%D0%B8%D0%B9_%D1%83%D0%BD%D1%96%D0%B2%D0%B5%D1%80%D1%81%D0%B8%D1%82%D0%B5%D1%82_%D1%96%D0%BC%D0%B5%D0%BD%D1%96_%D0%86%D0%B2%D0%B0%D0%BD%D0%B0_%D0%A4%D1%80%D0%B0%D0%BD%D0%BA%D0%B0">Львівського державного університету імені Івана Франка</a> (1980-1985), інженером-психологом відділу кадрів Львівського виробничого об&#8217;єднання «Іскра» (1985-1986), психологом служби сім&#8217;ї Львівської міської фірми обрядових послуг (1986-1989), викладачем <a title="Київський національний лінгвістичний університет" href="https://uk.wikipedia.org/wiki/%D0%9A%D0%B8%D1%97%D0%B2%D1%81%D1%8C%D0%BA%D0%B8%D0%B9_%D0%BD%D0%B0%D1%86%D1%96%D0%BE%D0%BD%D0%B0%D0%BB%D1%8C%D0%BD%D0%B8%D0%B9_%D0%BB%D1%96%D0%BD%D0%B3%D0%B2%D1%96%D1%81%D1%82%D0%B8%D1%87%D0%BD%D0%B8%D0%B9_%D1%83%D0%BD%D1%96%D0%B2%D0%B5%D1%80%D1%81%D0%B8%D1%82%D0%B5%D1%82">Київського державного педагогічного інституту іноземних мов</a> (1992-1993). В 1992 році після закінчення аспірантури <a title="Інститут психології імені Г. С. Костюка НАПН України" href="https://uk.wikipedia.org/wiki/%D0%86%D0%BD%D1%81%D1%82%D0%B8%D1%82%D1%83%D1%82_%D0%BF%D1%81%D0%B8%D1%85%D0%BE%D0%BB%D0%BE%D0%B3%D1%96%D1%97_%D1%96%D0%BC%D0%B5%D0%BD%D1%96_%D0%93._%D0%A1._%D0%9A%D0%BE%D1%81%D1%82%D1%8E%D0%BA%D0%B0_%D0%9D%D0%90%D0%9F%D0%9D_%D0%A3%D0%BA%D1%80%D0%B0%D1%97%D0%BD%D0%B8">Інституту психології АПН України</a> захистив кандидатську дисертацію на тему «Особливості статево-рольового самовизначення хлопчиків-підлітків із неповної сім&#8217;ї».</p><p>З 1994 року професійна діяльність В. О. Васютинського пов’язана з <a title="Інститут соціальної та політичної психології НАПН України" href="https://uk.wikipedia.org/wiki/%D0%86%D0%BD%D1%81%D1%82%D0%B8%D1%82%D1%83%D1%82_%D1%81%D0%BE%D1%86%D1%96%D0%B0%D0%BB%D1%8C%D0%BD%D0%BE%D1%97_%D1%82%D0%B0_%D0%BF%D0%BE%D0%BB%D1%96%D1%82%D0%B8%D1%87%D0%BD%D0%BE%D1%97_%D0%BF%D1%81%D0%B8%D1%85%D0%BE%D0%BB%D0%BE%D0%B3%D1%96%D1%97_%D0%9D%D0%90%D0%9F%D0%9D_%D0%A3%D0%BA%D1%80%D0%B0%D1%97%D0%BD%D0%B8">Інститутом соціальної та політичної психології НАПН України</a> (до 1996 року &#8211; Науково-практичним центром політичної психології). Він працював завідувачем лабораторії психології мас та організацій (1994-1998), провідним науковим співробітником лабораторії методології психосоціальних і політико-психологічних досліджень (2001-2007), був докторантом цього інституту (1998-2001). З 2007 року працює завідувачем лабораторії психології мас та спільнот. В 2006 році захистив докторську дисертацію на тему «Інтеракційна психологія влади».</p><p>&nbsp;</p></div></section><section class='section'><h2>Нагороди</h2><div><ul><li>Цінний подарунок <a title="Президент України" href="https://uk.wikipedia.org/wiki/%D0%9F%D1%80%D0%B5%D0%B7%D0%B8%D0%B4%D0%B5%D0%BD%D1%82_%D0%A3%D0%BA%D1%80%D0%B0%D1%97%D0%BD%D0%B8">Президента України</a> (2005)</li><li><a class="new" title="Медаль «Ушинський К.
