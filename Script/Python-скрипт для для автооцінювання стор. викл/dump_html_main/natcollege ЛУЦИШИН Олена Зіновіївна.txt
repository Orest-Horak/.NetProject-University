Олесько Буського району Львівської області. У 1986-1997 рр. навчалася в Олеській середній школі. З 1997 по 2002 рік навчалася на спеціальності біогеографія і географія ґрунтів географічного факультету Львівського національного університету імені Івана Франка. Здобула кваліфікацію географ-ґрунтознавець, викладач. З 2002 по 2005 рр. працювала на посаді вчителя географії в СЗОШ № 92 м. Львова. З 2005 по 2007 рр. працювала на посаді інженера в Природничому коледжі, з 2007 р. переведена на посаду викладача. У 2014 р. захистила дисертацію для здобуття наукового ступеня кандидат географічних наук на тему: &#8221;Ґрунти Надсянської рівнини: еколого-географічний аналіз&#8221;.</p></div></section><section class='section'><h2>Розклад</h2><div><p><a href="http://natcollege.lnu.edu.ua/about/staff/schedule-of-lecturers/?subpage=lecturer&id=8268&semester=1">1-й семестр</a><br /><a href="http://natcollege.lnu.edu.ua/about/staff/schedule-of-lecturers/?subpage=lecturer&id=8268&semester=2">2-й семестр</a></p></div></section></article>
