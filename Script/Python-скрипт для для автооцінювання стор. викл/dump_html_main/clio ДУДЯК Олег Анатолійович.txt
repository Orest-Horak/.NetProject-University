У 1991 р. закінчив Підбірцівську СШ Пустомитівського р-ну Львівської обл. Впродовж 1991-1996 рр. навчався на історичному факультеті Львівського державного університету імені Івана Франка. З 1995 по 1998 рр. працював вчителем історії Борщовицької СШ Пустомитівського р-ну Львівської обл.</p><p class="paragraph_style_3" style="text-align: justify;">У 1998 р. вступив на навчання до аспірантури Львівського державного університету імені Івана Франка при кафедрі історії слов’янських країн, яку закінчив у 2001 р. Дисертацію на тему “Динаміка соціальної структури і зайнятості польського населення Західної України у міжвоєнний період (20-30-і рр. ХХ ст.)” захистив у грудні 2003 р.</p><p class="paragraph_style_3" style="text-align: justify;">З липня 2003 р. працює асистентом кафедри давньої історії України та спеціальних історичних дисциплін. <span class="style_2">З січня 2011 р. на посаді доцента кафедри.</span></p><p class="paragraph_style_3" style="text-align: justify;">У жовтні 2012 р. присвоєно вчене звання доцента кафедри.</p><p class="paragraph_style_3" style="text-align: justify;">У червні 2015 р. &#8211; серпні 2016 р. перебував у лавах ЗСУ.</p></div></section><section class='section'><h2>Проекти</h2><div><p><a href="http://clio.lnu.edu.ua/wp-content/uploads/2015/01/1.2005-2008.-Західноукраїнські-землі-в-епоху-пізнього-середньовіччя-та-ранньомодерної-доби-2006-6.doc">1.2005-2008.
